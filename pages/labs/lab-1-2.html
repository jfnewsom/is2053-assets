The if-elif-else Chain

Use decision structures to set variables based on the player's choice.

An if-elif-else chain that sets 4 variables based on the choice: destination, distance, status, and message.

BookEx 3.4 (grader.py): Shows the if-elif-else pattern for handling multiple conditions. Each branch sets variables based on a score—you'll do the same based on a choice.

  1. AFTER getting the choice, start your if-elif-else chain
  2. For choice == 1 (Corpus): Set destination, distance (use constant), status to 'BEACH', and a fun message
  3. For choice == 2 (Houston): Set the same 4 variables with Houston values, status 'PROGRESS'
  4. For choice == 3 (Austin): Set the same 4 variables with Austin values, status 'DETOUR'
  5. For else (invalid): Set destination to 'Nowhere', distance to 0, status to 'ERROR'

Variables to set in each branch:

Did You Make It?

Calculate remaining miles and check if the player arrived.

An if-else that checks if miles_remaining is >= 0 (made it!) or < 0 (fell short). Use abs() to show how many miles short.

"Remember absolute value from math class? The notation |−50| = 50—it gives you the distance from zero, always positive. Python's abs() function does the same thing: abs(-50) gives you 50. When the player falls short, use it to show how many miles they missed by as a positive number. Note: abs() isn't in the Gaddis textbook, but your professor is introducing it here—so it's fair game for this assignment."

Project Manager, Bat City Collective
  1. Calculate remaining miles: miles_remaining = miles_rolled - distance
  2. Display "Heading to {destination}!" and the distance
  3. Check if made it: Write an if statement checking if miles_remaining >= 0
  4. If yes (made it): Print arrival message, display the status message variable, show miles to spare
  5. If no (fell short): Calculate miles_short using abs(), then print how many miles short

BookEx 3.2.2 (auto_repair_payroll.py): Shows if-else with different calculations in each branch. Same idea here—different output depending on whether you made it or not.
