Did You Make It?
What You're Building

An if-else that checks if miles_remaining is >= 0 (made it!) or < 0 (fell short). Use abs() to show how many miles short.

"Remember absolute value from math class? The notation |-50| = 50 gives you the distance from zero, always positive. Python's abs() function does the same thing: abs(-50) gives you 50. When the player falls short, use it to show how many miles they missed by as a positive number. Note: abs() isn't in the Gaddis textbook, but your professor is introducing it here — so it's fair game for this assignment."
Priya Venkatesh
Project Manager, Bat City Collective
BookEx Patterns
  • Program 3-2 — auto_repair_payroll.py — Shows if-else with different calculations in each branch. Same idea here — different output depending on whether you made it or not.
Step-by-Step Instructions
  1. Calculate remaining miles: miles_remaining = miles_rolled - distance
  2. Display "Heading to {destination}!" and the distance
  3. Check if made it: write an if statement checking if miles_remaining >= 0
  4. If yes (made it): print arrival message, display the status message variable, show miles to spare
  5. If no (fell short): calculate miles_short using abs(), then print how many miles short
Tips & Pitfalls
Mistake Why It Fails
Comparing wrong variable Check miles_remaining, not miles_rolled or distance
Forgetting abs() for miles_short Shows a negative number like "short by -50 miles"
Using > instead of >= Exact arrival (0 remaining) counted as a failure
