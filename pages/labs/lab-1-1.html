Setup & Welcome Banner

Create your file with the docstring, define constants, and display the welcome banner.

The foundation: a Python file with proper documentation, 4 distance constants, and a game banner.

Named constants make your code readable and maintainable. If a distance changes, you update ONE line instead of hunting through your entire program.

  1. Create a new file called lab-1-1.py
  2. Add your docstring at the very top with your name, abc123, section, and a description
  3. Define 4 constants for the distances from San Antonio:
# Constants - distances from San Antonio (in miles)
SAT_TO_CORPUS = 150
SAT_TO_HOUSTON = 200
SAT_TO_AUSTIN = 80

# Distance from Houston to Galveston
HOU_TO_GALVESTON = 50
  1. Add the main() function structure below your constants. All programs in this course use this pattern—copy it exactly (you'll learn why in Chapter 5):
def main():
    """Main function."""
    # Your code goes here (indented 4 spaces)
    pass  # Remove this line once you add real code


if __name__ == '__main__':
    main()

⚠ Important: From now on, ALL your code (except constants and imports) goes inside the main() function, indented 4 spaces. The if __name__ block always stays at the very bottom of your file.

  1. Inside main(), create a welcome banner using print() and string multiplication (notice the 4-space indent):
    # Welcome banner
    print()
    print('=' * 50)
    print('DEEP IN THE HEART: A LONE STAR JOURNEY')
    print('=' * 50)
    print()

BookEx 2.8 (string_multiplication.py): Shows how '=' * 50 creates a string of 50 equal signs. Perfect for banners!

Calculations

Calculate miles from the roll, then calculate remaining miles to each destination.

The math engine! Convert the roll to miles, then subtract from each destination distance to see how far you'd still need to travel.

BookEx 2.7.1 (simple_math.py): Shows the pattern of storing calculation results: pay = salary + bonus. You'll do similar math here.

  1. Calculate miles from the roll: Multiply the roll by 10 and store in miles_rolled
  2. Calculate remaining miles to Corpus: Subtract miles_rolled from SAT_TO_CORPUS
  3. Calculate remaining miles to Houston: Same pattern, different constant
  4. Calculate remaining miles to Austin: Same pattern, different constant
  5. Calculate Galveston total first: Add SAT_TO_HOUSTON + HOU_TO_GALVESTON
  6. Calculate remaining to Galveston: Subtract miles_rolled from the total

Step 1: Add debug lines AFTER your calculations:

    # DEBUG - DELETE AFTER TESTING

    print(f'DEBUG: corpus_remaining = {corpus_remaining}')
    print(f'DEBUG: galveston_total = {galveston_total}')
