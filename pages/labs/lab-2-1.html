Complete the Loop Body
What You're Building

The complete loop body that: shows turn info, travels 50 miles, updates miles_remaining so the loop eventually ends, accumulates total_miles, and increments the turn counter.

Why This Matters

The accumulator pattern (total_miles = total_miles + MILES_PER_TURN) is one of the most common shapes in all of programming. Shopping carts add prices. Game scores tally points. Fitness apps count steps. Every running total in every program you'll ever use is some version of this exact line.

Step-by-Step Instructions

Inside your while loop (indented!), do these things in order:

  1. Display the turn header — show what turn number this is
  2. Display miles remaining — so the player sees their progress
  3. Display a travel message — "You travel 50 miles..." (use your constant!)
  4. Update miles_remaining — SUBTRACT miles traveled
  5. Update total_miles — ADD miles traveled (accumulator pattern!)
  6. Increment turn — add 1 so next iteration shows Turn 2, Turn 3, etc.
