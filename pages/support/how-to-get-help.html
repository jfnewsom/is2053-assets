Ask on Discord
Fastest Response Time
Why Discord?

The class Discord is the fastest way to get help. Post your question in #help-python and classmates or the instructor will respond—often within minutes. You also get the benefit of seeing how other students have solved similar problems.

How to Ask a Good Question

The better you describe your problem, the faster you’ll get an answer.

  • Describe what you’re trying to do — what is the goal of this code?
  • Share the error message — paste it in a code block so it’s readable
  • Show what you’ve already tried — this helps skip suggestions you’ve already ruled out
  • Be specific — “it doesn’t work” doesn’t help us help you
lightbulb
Using Code Blocks

Wrap your code and error messages in triple backticks so Discord formats them correctly. Type ```, paste your code, then ``` again. This makes your question much easier to read.
