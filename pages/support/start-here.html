Overview

Welcome to IS2053! Before diving into Python, let’s make sure you’re fully set up for success. Complete each checkpoint below in order—they’re designed to build on each other. Most students finish in 70–100 minutes.

“Hey there! I’m Danny, and I’ll be your guide through this course. Don’t rush through these setup steps—a solid foundation saves hours of frustration later. If you hit any snags, swing by our Tuesday Zoom session. We’re here to help!”

Lead Developer, Bat City Collective
What You’re Building

By the end of these checkpoints, you’ll have a complete Python development environment ready to go. You’ll be introduced to the course narrative, meet your mentors at Bat City Collective, and write your very first Python program. This is your launchpad for everything that comes next!

Why This Matters

Professional developers spend real time configuring their tools. Skipping setup steps or rushing through them leads to mysterious errors later. Taking time now to understand your environment will make every assignment smoother.

Program Requirements
  • Python 3.12+ installed with “Add to PATH” checked
  • VS Code installed with the Python extension
  • Flake8 extension installed for code style checking
  • IS2053 folder created and opened in VS Code
  • hello.py program runs successfully
  • CodeGrade payment method ready (credit card or bookstore code)
  • Discord & Discussion Forum access confirmed

CHECKPOINT
4
Install VS Code
5 minutes — Your code editor

Download Visual Studio Code from code.visualstudio.com. This is our code editor for the semester—it’s free, powerful, and used by professional developers worldwide.

Download VS Code →

CHECKPOINT
5
Install the Python Extension
2 minutes — Syntax highlighting and run button

In VS Code, click the Extensions icon (or press Ctrl+Shift+X). Search for “Python” and install the one by Microsoft. This gives you syntax highlighting, error detection, and the run button.

Open in VS Code Marketplace →

CHECKPOINT
6
Install the Flake8 Extension
3 minutes — Code style checker (graded!)

Still in Extensions, search for “Flake8” and install the one by Microsoft. Flake8 checks your code style—things like consistent spacing, proper indentation, and unused variables.

error
This Is Graded

CodeGrade uses Flake8 to check your code style on every submission. Installing it now means you’ll see the same warnings before you submit, so you can fix them and earn full points.

lightbulb
How It Works

After installing, you’ll see yellow/red squiggly lines under style issues in your code. Hover over them to see what’s wrong and how to fix it. We have a full Flake8 Guide in Canvas that explains every error code.

Open in VS Code Marketplace →

CHECKPOINT
7
Create Your Workspace Folder
2 minutes — Your IS2053 folder

Create a folder on your computer called IS2053. This is where all your code will live for the semester. In VS Code, go to File → Open Folder and select it.

lightbulb
Pro Tip

Put it somewhere easy to find, like your Documents folder. Avoid cloud-synced folders (OneDrive, Dropbox) as they can cause file-locking issues.

CHECKPOINT
8
Write Your First Program
5 minutes — Your first Python program!

Create a new file called hello.py and type:

print("Hello, World!")

Click the Play button (top right) to run it. You should see Hello, World! appear in the terminal below.
