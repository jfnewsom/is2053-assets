Install Python 3.12+

Download Python from python.org. Get version 3.12 or newer.

CRITICAL: During installation, check “Add Python to PATH”. Missing this step causes headaches later!

GO! →

Install VS Code

Download Visual Studio Code from code.visualstudio.com. This is our code editor for the semester—it’s free, powerful, and used by professional developers worldwide.

GO! →

Install the Python Extension

In VS Code, click the Extensions icon (or press Ctrl+Shift+X). Search for “Python” and install the one by Microsoft. This gives you syntax highlighting, error detection, and the run button.

GO! →

Install the Flake8 Extension

Still in Extensions, search for “Flake8” and install the one by Microsoft. Flake8 checks your code style—things like consistent spacing, proper indentation, and unused variables.

THIS IS GRADED! CodeGrade uses Flake8 to check your code style on every submission. Installing it now means you’ll see the same warnings before you submit, so you can fix them and earn full points.

How it works: After installing, you’ll see yellow/red squiggly lines under style issues in your code. Hover over them to see what’s wrong and how to fix it. We have a full Flake8 Guide in Canvas that explains every error code.

GO! →

Create Your Workspace Folder

Create a folder on your computer called IS2053. This is where all your code will live for the semester. In VS Code, go to File → Open Folder and select it.

Pro Tip: Put it somewhere easy to find, like your Documents folder. Avoid cloud-synced folders (OneDrive, Dropbox) as they can cause file-locking issues.
