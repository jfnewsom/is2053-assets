Python Programming • Spring 2026 • UT San Antonio

Welcome to IS2053 Programming I! This semester, you'll learn Python by building a complete video game for a (fictional) client. No prior programming experience required—just curiosity and a willingness to learn.

Our textbook is Starting Out with Python, 6th Edition by Tony Gaddis. We'll cover Chapters 2-11 across five modules, each building on the last.

Module 4 Chapters 8-9 File I/O, exception handling, dictionaries
Module 5 Chapters 10-11 Classes, objects, inheritance
