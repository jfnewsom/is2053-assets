Welcome

This semester you’ll learn Python by building a complete video game for a fictional client. No prior programming experience required — just curiosity and a willingness to work through problems one step at a time.

Our textbook is Starting Out with Python, 6th Edition by Tony Gaddis. We cover Chapters 2–11 across five modules, each building on the last. Every lab adds to the same codebase — by the end of the semester you’ll have a working game.

Course Overview
5 Modules • Build → Enhance → Master

Five modules, three weeks each. Every module follows the same rhythm: Build (new concepts), Enhance (add features), and Master (polish and complete). You’re always working on the same program — each lab builds on the last.

Module Chapters Topics Weeks
Module 1 Getting Started CH 2–3 Variables, input/output, calculations, strings, decision structures
Build Enhance Master
Module 2 The Journey Begins CH 4–5 Loops (for/while), nested loops, input validation, math module
Build Enhance Master
Module 3 Keeping Track CH 6–7 Functions, modular program design, lists, basic list processing
Build Enhance Master
Module 4 Smarter Code CH 8–9 File I/O, exception handling, sets, dictionaries
Build Enhance Master
Module 5 Building Like Professionals CH 10–11 Classes, objects, inheritance, extending functionality
Build Enhance Master
