Welcome

This semester you’ll learn Python by building a complete video game for a fictional client. No prior programming experience required — just curiosity and a willingness to work through problems one step at a time.

Our textbook is Starting Out with Python, 6th Edition by Tony Gaddis. We cover Chapters 2–11 across five modules, each building on the last. Every lab adds to the same codebase — by the end of the semester you’ll have a working game.

Meet Your Instructor
John Newsom
Professor • Information Systems & Cyber Security

I’ve been teaching programming at UT San Antonio since 2020. Before academia, I spent 15 years in industry as an IT manager at small companies — ad agencies and law firms — doing everything from database work to full-stack development.

My teaching philosophy: “Build a little, test it, fix it.” We tackle Python one concept at a time, always with hands-on practice.

Course Overview
5 Modules • Build → Enhance → Master

Five modules. Every module follows the same rhythm (Module 5 is two weeks): Build (new concepts), Enhance (add features), and Master (polish and complete). You’re always working on the same program — each lab builds on the last.

Module Week • Type • Content
Module 1 Getting Started CH 2–3 Build Week 01 • CH 2 Variables, input/output, f-strings • Lab 1-1
Enhance Week 02 • CH 3 Decision structures, if/elif/else • Lab 1-2
Master Week 03 • Lab 1-3 • Module 1 Quiz
Module 2 The Journey Begins CH 4–5 Build Week 04 • CH 4 for/while loops, nested loops, input validation • Lab 2-1
Enhance Week 05 • CH 5 Functions, random module, function_library.py • Lab 2-2
Master Week 06 • Lab 2-3 • Module 2 Quiz
Module 3 Keeping Track CH 6–7 Build Week 07 • CH 6 File I/O, exception handling • Lab 3-1
Enhance Week 08 • CH 7 Lists, tuples, list processing • Lab 3-2
Master Week 09 • Lab 3-3 • Module 3 Quiz
Module 4 Smarter Code CH 8–9 Build Week 10 • CH 8 String methods, file-driven data • Lab 4-1
Enhance Week 11 • CH 9 Dictionaries, sets, data structures • Lab 4-2
Master Week 12 • Lab 4-3 • Module 4 Quiz
Module 5 Building Like Professionals CH 10–11 Build Week 13 • CH 10 Classes, objects, OOP fundamentals • Lab 5-1
Enhance Week 14 • Lab 5-2 • Final Module Quiz
