Troubleshooting
"No tests ran"
Your filename is wrong. Check that it matches exactly (case-sensitive, hyphens not underscores).
"Output doesn't match"
Check for extra spaces, missing punctuation, or wrong capitalization. Output must match exactly.
"Scope violation detected"
You used a technique we haven't covered yet. Check the assignment for what's allowed.
"Flake8 errors"
Code style issues. Check line length (max 79 characters), spacing around operators, etc.
"Program timed out"
Your code has an infinite loop. Check your while loop conditions.
