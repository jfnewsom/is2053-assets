Module 1: Getting Started
Units 1–3 • Chapters 2 & 3 • All My Eggses Live in Texas

Welcome to Bat City Collective! Your first module establishes your development environment and introduces the foundations of Python programming. You’ll build the character creation screen for Deep in the Heart: A Lone Star Journey—the Texas road trip game you’ll develop all semester for our restaurant client.

This Module
Unit Assignment Focus
Unit 1 BUILDLab 1-1: Welcome to San Antonio Variables, input/output, f-strings. Build the character creation screen.
Unit 2 ENHANCELab 1-2: Choose Your Destination Decision structures (if/elif/else). Player picks a city and gets feedback.
Unit 3 MASTERLab 1-3: Hit the Road Nested decisions, logical operators. Complete first trip with outcomes.
Client Context: All My Eggses Live in Texas

What the Client Needs This Module

Bobby Hernandez (Director of Technology) is replacing the peg games on restaurant tables with tablet terminals. He wants a retro Texas road trip game that families can play while waiting for their chicken fried steak. Your job: build the first playable prototype.

“Before you build a castle, build a room. Before a room, build a wall. Before a wall, place one brick. That’s Module 1—we’re laying bricks.”

Danny Cavanaugh
Lead Developer, Bat City Collective
Skills & End State

What You’ll Build and Learn

Skills You’ll Build
  • Declare and use variables
  • Get user input with input()
  • Format output with f-strings
  • Use if/elif/else structures
  • Apply logical operators (and/or)
  • Write clean, readable code
End State

By the end of Module 1, your game will: create a character, pick a destination from San Antonio, and show different outcomes based on the player’s choices—all deterministic, no randomness yet. It’s a complete experience, just simple.
