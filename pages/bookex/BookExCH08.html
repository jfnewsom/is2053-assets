count_Ts.py
Program 8-1 (6th Ed) • Program 8-1 (5th Ed)
What You're Building

A program that asks the user to type a sentence, then loops through every character in the string and counts how many times the letter T (uppercase or lowercase) appears. The for ch in string pattern is the standard way to visit every character one at a time.

Step-by-Step Instructions
  1. Open your textbook to Program 8-1.
  2. Create a new file named count_Ts.py.
  3. Type the program exactly as shown, including the def main() and the if __name__ == '__main__': guard.
  4. Run it and type a sentence containing several T's.
  5. Confirm the count matches what you expect.
The or Checks Both Cases at Once

The condition if ch == 'T' or ch == 't': catches both uppercase and lowercase in one check. You can't write if ch == 'T' or 't': — that always evaluates to True because a non-empty string is truthy.

concatenate.py
Program 8-2 (6th Ed) • Program 8-2 (5th Ed)
What You're Building

A program that creates a string variable, prints it, then concatenates a last name to it using + and reassigns the result back to the same variable. This demonstrates that strings are immutable — concatenation creates a new string and the variable is updated to point to it.

Step-by-Step Instructions
  1. Open your textbook to Program 8-2.
  2. Create a new file named concatenate.py.
  3. Type the program including the def main() structure.
  4. Run it and confirm both print statements produce the expected output.
Strings Are Immutable

You can't change a string in place. name = name + ' Brown' creates a brand new string and points the variable name at it. The original 'Carmen' still exists briefly in memory before being garbage collected.

What You're Building

A program that asks the user to enter a string, then runs it through six testing methods — isalnum(), isdigit(), isalpha(), isspace(), islower(), and isupper() — and prints a message for each one that returns True. Multiple messages can print for the same input.

These Are All Independent if Statements

This program uses six separate if statements, not if-elif. That means multiple messages can print for the same input. For example, 'abc' prints both the alphanumeric and the alphabetic messages.

string_split.py
Program 8-9 (6th Ed) • Program 8-9 (5th Ed)
What You're Building

A program that creates a string with multiple words, calls split() with no arguments to break it on whitespace, and prints the resulting list. This is the simplest form of split() and the foundation for any program that needs to process words individually.

Step-by-Step Instructions
  1. Open your textbook to Program 8-9.
  2. Create a new file named string_split.py.
  3. Type the program including the def main() structure.
  4. Run it and confirm the output shows a list of four words.
split() Returns a List

split() with no arguments splits on any whitespace (spaces, tabs, newlines) and returns a list of strings. Multiple spaces between words are treated as one separator — no empty strings appear in the result.

split_date.py
Program 8-10 (6th Ed) • Program 8-10 (5th Ed)
What You're Building

A program that stores a date string in MM/DD/YYYY format, calls split('/') to break it on the slash character, and then accesses each part by index to print the month, day, and year separately. This is the core technique for parsing any delimited data.

Step-by-Step Instructions
  1. Open your textbook to Program 8-10.
  2. Create a new file named split_date.py.
  3. Type the program including the def main() structure.
  4. Run it and confirm the month, day, and year print on separate lines with labels.
  5. Try changing the date string to a different date and re-run.
The Delimiter Is Removed from the Results

'11/26/2023'.split('/') gives ['11', '26', '2023'] — the slashes are gone. Index 0 is month, 1 is day, 2 is year. This same pattern works for comma-separated data, file paths, or any structured string.
