Program 9-3: The .get() Method
What You're Building

Program 9-3 introduces .get(), which returns a default value when a key doesn't exist instead of raising a KeyError.

Step-by-Step Instructions
  1. Type Program 9-3 from the textbook.
  2. Run it once with a key that exists and once with a key that does not.
  3. Note the difference in output.
Direct Access vs .get()

Using my_dict['missing_key'] raises a KeyError. Using my_dict.get('missing_key', 'default') returns the default safely.

Program 9-4: Sets and Sorted Output
What You're Building

Program 9-4 introduces sets — unordered collections of unique values. The sorted() function is used to produce consistent output for grading.

Step-by-Step Instructions
  1. Type Program 9-4 from the textbook.
  2. Run it and observe that duplicates are eliminated automatically.
  3. Wrap the set in sorted() when printing to get consistent ordering.
Always Use sorted() for Grading

Sets are unordered. CodeGrade compares your output to expected output line by line. Always print sets using sorted() to guarantee consistent ordering.
