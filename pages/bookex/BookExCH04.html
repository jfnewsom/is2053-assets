lightbulb
range(5) Generates 0 Through 4

range(5) produces 0, 1, 2, 3, 4 — five values starting at zero. The loop variable x gets each of these, but since the print statement doesn't use x, the same message prints five times.

lightbulb
Why END_SPEED Is 131, Not 130

range() excludes the end value. If END_SPEED were 130, the last value generated would be 120. Setting it to 131 ensures 130 itself is included in the table.

Input Validation Is a Core Pattern

The inner validation loop — get input, then while it's invalid display an error and re-prompt — is a pattern you will use in almost every Lab assignment. Get comfortable with this structure now.

test_score_averages.py
Program 4-22 (6th Ed) • Program 4-17 (5th Ed)
What You're Building

A grade calculator that uses nested for loops to process multiple students, each with multiple test scores. The outer loop iterates through students; the inner loop collects scores and accumulates the total. The average is calculated and printed after the inner loop completes for each student.

Step-by-Step Instructions
  1. Open your textbook to Program 4-22 (6th Ed) or Program 4-17 (5th Ed).
  2. Create a new file named test_score_averages.py.
  3. Type the program carefully, paying attention to indentation at each loop level.
  4. Run it with 2 students and 3 test scores each as a first test.
  5. Confirm each student's average is correct and that the accumulator resets between students.
warning
Reset the Accumulator Inside the Outer Loop

total = 0.0 must be inside the outer loop but outside the inner loop. Before the outer loop: only resets once. Inside the inner loop: resets after every score. The right place is at the start of each student's processing — indented under the outer for, above the inner for.
