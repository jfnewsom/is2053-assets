lightbulb
:.2f Means Two Decimal Places

The colon starts the format spec, .2 sets precision to 2 decimal places, and f means floating-point. The entire spec goes inside the curly braces: {monthly_payment:.2f}.

columns.py
Program 2-23 (6th Ed) • Program 2-23 (5th Ed)
What You're Building

A program that prints six decimal values arranged in two right-aligned columns using the :10.2f format specifier. The 10 reserves a field of 10 characters; the .2f rounds to two decimal places. The result is a clean, aligned table.

Step-by-Step Instructions
  1. Open your textbook to Program 2-23.
  2. Create a new file named columns.py.
  3. Type the code carefully, paying close attention to the format specifiers.
  4. Run the program and verify the numbers form neat aligned columns.
  5. Compare your output to the book to confirm alignment is correct.
lightbulb
Field Width Controls Spacing

The number before the decimal in the format spec (10 in :10.2f) is the field width. All values right-align within that field, which is what creates the column effect.

Checkpoint
10
center_align.py
Program 2-24 (6th Ed) • Program 2-24 (5th Ed)
What You're Building

A program that prints six names, each centered in a 20-character field using the :^20 format specifier. The *** markers on each side let you see exactly how wide the field is and confirm the text is centered.

Step-by-Step Instructions
  1. Open your textbook to Program 2-24.
  2. Create a new file named center_align.py.
  3. Type the example and confirm the output is centered correctly.
  4. Try changing :^20 to :<20 (left) and :>20 (right) to see the difference.
info
Three Alignment Specifiers

^ centers text, < left-aligns text, and > right-aligns text (the default for numbers). All three go inside the curly braces before the field width number.
