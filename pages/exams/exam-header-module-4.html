Tips for Success
  • Strings are immutable — Methods return new strings, don't modify in place
  • Dictionary access — dict['key'] vs dict.get('key', default)
  • Keys must be unique — Duplicate keys overwrite previous values
  • Sets have no duplicates — {1, 2, 2, 3} becomes {1, 2, 3}
  • Know your set operations — Union (|), intersection (&), difference (-)
