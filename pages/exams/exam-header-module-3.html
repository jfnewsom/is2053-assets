Tips for Success
  • Know file modes — 'r' vs 'w' vs 'a' and what each does
  • Understand try/except — What happens when exceptions are caught?
  • List indexing starts at 0 — list[0] is the first element!
  • Slicing excludes the end — list[1:4] gives indices 1, 2, 3
  • Mutability matters — Lists can be modified; strings cannot
